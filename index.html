About

  • This application was made using the city of Ottawa's open data tree inventory to show unique information about the trees owned and cared for by the city.

    The trees generated display the color of the current foliage; the size of the tree on the map is based on a calculation combining the species growth factor, the diameter at breast height (DBH), and estimated age for that tree.

    Selecting a list will populate the map with the subset of trees in that category. Please be patient! The data and calculations can take some time to load.
